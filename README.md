This repository was created to practice basic Git and GitHub commands.

- Created a public GitHub repository.
- Added a `main.cpp` file.
- Made the initial commit.
- Cloned the repository to my local computer.
- Checked the repository status using `git status`.
- Modified the `main.cpp` file by updating:
  - Name
  - Student ID
  - Section
  - Additional information
- Viewed the changes using `git diff`.
- Staged the changes using `git add`.
- Committed the changes with a meaningful commit message.
- Pushed the updated project to GitHub.

- Create a GitHub repository.

- Track file changes.
